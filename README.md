# Permissor – CMP template for Google Tag Manager

A Google Tag Manager custom tag template that integrates the **Permissor**
Consent Management Platform (CMP) with **Google Consent Mode v2** and the
**IAB TCF v2.2** framework.

To use this template, create an account on [Permissor](https://permissor.it) and follow the instructions for implementation: https://permissor.it/cmp-google
